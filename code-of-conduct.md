# Code of Conduct

## Our Goal

This project exists to share useful, practical, and high-quality guidelines.  
We want to keep this space constructive, respectful, and focused on value.

---

## Expected Behavior

- Be respectful and professional in all interactions
- Focus on ideas, not individuals
- Provide constructive feedback, not criticism without value
- Be open to different perspectives and experiences
- Assume good intent, but prioritize clarity and accountability

---

## Unacceptable Behavior

- Harassment, discrimination, or hate speech in any form
- Personal attacks, insults, or toxic behavior
- Trolling or intentionally disruptive actions
- Sharing private or sensitive information without permission
- Repeated low-quality or non-constructive contributions

---

## Maintainer Responsibility

The maintainer is responsible for keeping the project aligned with its goals.

Actions that may be taken include:

- Requesting changes to contributions
- Removing comments, issues, or pull requests
- Limiting or blocking participation if necessary

Decisions are based on maintaining a healthy and productive environment.

---

## Scope

This Code of Conduct applies to:

- GitHub issues, pull requests, and discussions
- Any interaction related to this project

---

## Reporting

If you experience or observe behavior that violates this Code of Conduct, you can report it to:

📧 w.kristories@gmail.com

---

## Final Note

This is a focused, guideline-driven project.  
The goal is not strict formality, but a clean, respectful, and high-signal environment.
